# Orbit - Focus & Explore

A productivity app by **Bright Stars** that gamifies reducing phone usage through space exploration.

## What is Orbit?

Orbit helps teens and university students limit excessive phone use by turning focus time into a space adventure. Stay focused, complete missions, and build your galaxy!

### Features

- **Focus Timer** - Start focus sessions (15, 30, 45, or 60 min). A planet builds as you stay focused — leave early and it's destroyed.
- **Missions** - Complete daily productivity tasks to earn extra XP and credits.
- **Galaxy Map** - Unlock and explore new planets as you accumulate focus time.
- **Profile & Stats** - Track your level, streaks, total focus time, and achievements.
- **Points System** - Earn XP and Stardust credits for every completed session and mission.

### How It Works

1. Choose a focus duration and start a session
2. Stay in the app — your planet grows as the timer counts down
3. Complete the session to earn XP, credits, and discover new planets
4. Complete bonus missions for extra rewards
5. Build up your galaxy and climb the levels!

## Tech Stack

- HTML, CSS, JavaScript (single-file web app)
- No dependencies or build tools required
- Data saved locally via localStorage

## Getting Started

Just open `index.html` in any modern web browser. No setup needed!
